# shopify-estimated-delivery-date

Shopify で、インタラクティブな報酬獲得体験を実現できるようにするアプリ
ユーザーの CVR を上げることが目的

# 要件

- 「あなた限定」「本日限定」「クーポンプレゼント！」
- 「新規ユーザー特典」ルーレットが回る
  - 0
  - 700
  - 3000
  - 7000
  - 9300
  - 40000
- 流れ
  - 1 回目、「9300」で止まる →「やったね」
  - 2 回目、「0」で止まる →「残念」
  - 3 回目 今だけ、もう 1 回挑戦できるチャンス！→「押せ！！」→ ボタン連打 →「やったね！」
  - 4 回目、「40000」で止まる →「おめでとうございます！！大当たりです」
  - 合計 50000 円！
  - 「素晴らしい！99%のユーザーを超えました！」
  - 「やったね、ブラボー」などのコメントが画面に流れる
- 今から 20 分以内の購入で適用されます！
- 3,2,1 スタート！
- 画面上部に 20 分のカウントダウンの帯が表示される
